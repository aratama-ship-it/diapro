# ディアボロ育成ゲーム — イベント追加メモ

- 場所: `app-dev/diabolo-trainer/docs/ideas/`
- 位置づけ: [[project_diabolo_trainer_app]]の「アイデアタスク」枠に合流。既存EVENT_DRAFTS方式（data.js）で実装する想定。
  「大道芸人ジョージ」はキャラ+人物イベント＝[[project_diabolo_trainer_codex_review]]が挙げた次の面白さ「NPCと目標提示」の候補にあたる。

## 追加イベント候補

### 1. マレーシアの大会イベント
- 海外大会として発生
- 遠征費、移動、経験値、知名度などの要素と接続可能
- 国内大会とは異なる参加条件や報酬を設定する余地あり

### 2. 大谷はちみつ園イベント
- 大谷はちみつ園に関連する特別イベントとして発生
- 地域イベント、交流イベント、スポンサーイベント、特殊報酬イベントなどとして検討可能
- 詳細内容は後で設定

### 3. 大道芸人ジョージイベント
- 大道芸人ジョージに関連する人物イベントとして発生
- パフォーマンス、交流、助言、依頼、共演、技術習得などへ展開可能
- 詳細内容は後で設定

## 実装状況（2026-07-15 β実装）
3件とも `js/data.js` の `EVENTS` に実装済み（ランダム抽選のキャライベント/ハプニング枠）。数値はβ・要バランス調整。

1. **マレーシアの大会イベント** → `charEvents` の `malaysia_trip`（海外遠征の決断イベント）
   - 遠征する: 演技構成+5・新奇性+3・やる気+15・体力-22・学力-10 ／ 国内に専念: 学力+6・やる気-2
   - 台湾合宿(新奇性)と役割を分け、舞台度胸＝演技構成/やる気に振った。※現状は"遠征の決断"で、順位のつく本物の大会ではない（将来: 独立した海外大会ロジック化も可）
2. **大谷はちみつ園イベント** → `happenings` の `hap_honey`（地元スポンサー）
   - 差し入れで やる気+10・体力+8（回復）
3. **大道芸人ジョージイベント** → `CHARACTERS` に `george` 追加＋`charEvents` の `george1`/`george2`（NPC人物イベント）
   - george1: 見せ技を教わる=新奇性+3 ／ 飛び入り共演=演技構成+2・やる気+6
   - george2: ついて行く=演技構成+3・体力-6 ／ 練習優先=操作安定+2・やる気+4
