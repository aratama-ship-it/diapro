# ショート版・2ヶ月ターン実験

作成日: 2026-07-23

## 今回の解釈

ユーザー案の「偶数月は練習、奇数月はイベント」を次のように実装した。

- 内部の48ヶ月は維持し、大会・定期テスト・怪我・補習の月数を変えない。
- プレイヤーの1ターンは連続する2ヶ月。全24ターン。
- 4月・6月・8月…の偶数月に、練習・勉強・休養を1回選ぶ。
- 5月・7月・9月…の奇数月は、状態イベント・ランダムイベントを処理して自動で進む。
- 大会・世界大会・JJF・固定イベントは、本来設定されている月に処理する。偶数月の大会は行動後に開催される。
- 練習で確定した能力上昇値は2倍。疲労・怪我リスク・イベント効果は倍にしない。

通常版は `index.html`、ショート版は `short.html` または `index.html?mode=short`。セーブ、卒業記録、カード図鑑は別領域に保存する。

## 自動シミュレーション

コマンド:

```sh
npm run simulate:short -- 1000
```

条件は高校経験者、弱点優先の練習、試験前は学力45未満なら勉強、疲労55超なら休養。通常版とショート版をそれぞれ1000シード実行した。

## 判定

- **速度:** 狙いどおり、プレイヤーの選択回数は48回から24回へ半減した。
- **完走性:** ショート版は1000シードすべて卒業。通常版は1件だけ退学だったため99.9%。ショート版だから退学しやすい状態にはなっていない。
- **強さ:** 成長2倍だけでは通常版相当にならない。能力平均は約8.7低く、ポイントは約35%低い。
- **原因の仮説:** 勉強は9.5回から8.3回にしか減らず、短縮した24行動の中で占める比率が大きい。その結果、練習回数は通常版の半分より少ない。さらに、序盤大会までの練習機会が少ないため、後半で能力が追いついてもポイント差が残る。

## 次に比較する候補

今回の2倍版を基準として残し、次の一手は一度に1項目だけ変えて再比較する。

1. 練習ゲインは2倍のまま、勉強1回の学力ゲインも2倍にする。
2. または、ショート版だけ学力の自然低下を `-2/月` から `-1/月` にする。
3. 能力は現在のままでもよい場合、ショート版専用に卒業ランクのポイント境界だけ調整する。

「短時間で少し厳しい別ゲーム」を狙うなら、現状の2倍版にも独立した価値がある。通常版と同じ強さを狙うなら、単純な練習倍率より学業コストの調整を先に試すのが妥当。
