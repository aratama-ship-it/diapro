# ショート版・2ヶ月ターン実験

作成日: 2026-07-23

## 今回の解釈

ユーザー案の「偶数月は練習、奇数月はイベント」を次のように実装した。

- 内部の48ヶ月は維持し、大会・定期テスト・怪我・補習の月数を変えない。
- プレイヤーの1ターンは連続する2ヶ月。全24ターン。
- 4月・6月・8月…の偶数月に、練習・勉強・休養を1回選ぶ。
- 5月・7月・9月…の奇数月は、状態イベント・ランダムイベントを処理して自動で進む。
- 大会・世界大会・JJF・固定イベントは、本来設定されている月に処理する。偶数月の大会は行動後に開催される。
- 練習で確定した能力上昇値と、勉強で確定した学力上昇値は2倍。疲労・怪我リスク・イベント効果は倍にしない。

通常版は `index.html`、ショート版は `short.html` または `index.html?mode=short`。セーブ、卒業記録、カード図鑑は別領域に保存する。

## 自動シミュレーション

コマンド:

```sh
npm run simulate:short -- 1000
```

条件は高校経験者、弱点優先の練習、試験前は学力45未満なら勉強、疲労55超なら休養。通常版とショート版をそれぞれ1000シード実行した。同じシード列と意思決定方針を使い、試験までの先読みは通常版1ヶ月、ショート版2ヶ月とした。能力平均は12個の技術値と演技構成の訓13値の平均。

これは固定した自動行動方針によるバランス比較であり、人間のプレイヤーが感じる難易度や楽しさを直接測ったものではない。また、モード間で行動回数と乱数消費が異なるため、同一シードは同一プレイの1対1対応を意味しない。

## 判定

- **速度:** 狙いどおり、プレイヤーの選択回数は48回から24回へ半減した。
- **完走性:** ショート版は1000シードすべて卒業。通常版は1件だけ退学だったため99.9%。ショート版だから退学しやすい状態にはなっていない。
- **学業コスト:** 勉強を2倍にすると、勉強回数は8.3から5.1へ減り、練習回数は15.1から18.2へ増えた。狙いどおり、24行動の中で学業が占める割合が下がった。
- **強さ:** 第1実験と比べ、最終能力は62.3から67.7（+5.4）、ポイントは482.3から621.9（+139.6、約29%増）、優勝は1.12から1.92へ改善した。
- **怪我:** 練習回数が増えたため、平均怪我回数は0.86から1.03へわずかに増えた。ただし、通常版の2.12回よりは少ない。また、怪我後の奇数月で回復期間が進むため、ショート版の療養行動は0.0回。怪我発生はするが、次のプレイヤー行動を消費しない仕様になっている。
- **残る差:** 通常版と比べると、最終能力は3.3低く、ポイントは約16%低い。優勝数と世界大会出場数にも差がある。序盤大会までの練習機会が半減し、後半の追い上げでは既に失った大会ポイントを取り戻せないのが主な仮説。

## 現時点の採用案

「練習・勉強の上昇値2倍」をショート版の現行案とする。通常版とまったく同じ強さではないが、操作回数を半分にしつつ、「成長できない」感覚を大幅に緩和できる。最終能力は通常版に近づいたが、優勝数は約31%、世界大会出場数は約28%少なく、大会実績は明確に厳しい別ゲームである。また、療養行動が発生しない点は別軸の難易度緩和であり、仕様として残すかは実プレイで確認が必要。

今後、通常版とポイント帯まで揃えたい場合は、この現行案を基準に、序盤の大会補正かショート版専用の卒業ランク境界を1項目ずつ比較する。
