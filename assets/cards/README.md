# カード画像の保存区分

- `web/`: アプリ配布用。592x300、JPEG品質82。Git管理する。
- 直下のPNG: 採用画像の原本。iCloudワークスペースには残すが、配布サイズを抑えるためGit管理しない。
- `concepts/`: 比較・検討用の候補原本。iCloudワークスペースには残すがGit管理しない。
- `mx_B_innovator_provisional.jpg`: 「奇手の使い手」のギャラリー限定仮案。通常プレイには使用しない。

配布用画像は中央基準の `object-fit: cover` 相当で切り出している。採用画像を差し替える場合は、原本を残したうえで `web/` の同名JPEGを更新する。
